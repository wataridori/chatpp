Notices

■　If there are two or more emoticons in different data but have the same name, only one is used. It is the emoticon belongs to the data that have the highest priority.
■　Use the "Move Up" button to arrange your data priority. The data in upper place will have higher priority
■　You can create your own Emoticons data file and then share it with your friends. Of course if you and your friends have the same data, you will see the same emoticons on Chatwork!
